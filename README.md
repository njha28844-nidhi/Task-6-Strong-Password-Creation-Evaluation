## 6. Reason for Strength Differences

### Weak Passwords:
- Short length
- Only letters + numbers
- Easy to guess
- Can be cracked using dictionary attacks

### Strong Passwords:
- Longer length
- Mix of uppercase, lowercase, numbers and symbols
- Difficult to guess
- Resistant to brute force and dictionary attacks
